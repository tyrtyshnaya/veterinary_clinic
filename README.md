# veterinary_clinic
сайт для ветеринарной клиники "Ветеринар и К"
Веь-приложение ветеринарной клиники с новостями, услугами, каталогом товаров, корзиной, профилем и оформлением заказа.
Функционал:
- Просмотр услуг/каталога товаров
- Добвление в корзину
- Оформление заказа
- Оформление звонка для услуги
- Просмотр новосте
- Администраторская панель

Требования:
-PHP 7.4+
-MySQL
-Веб-сервер

Установка и запуск
1. Клонирование репозита git clone https://github.com/tyrtyshnaya/veterinary_clinic.git
2. Создание базы данных, например:
   vet_shop.sql
3. Импорт БД
  импортируйте файл: database.sql
4. Настройка подключения
   Создайте файл db_connect.php
   Пример содержимого:
    $host = "MySQL-8.4"; 
    $username = "root";
    $password = "";
    $database = "ваша база";
    
    $db = mysqli_connect($host, $username, $password, $database);
    
    mysqli_set_charset($db, "utf8mb4");

5. Запуск проекта
   Поместите проект в папку сервера: OpenServer -> home
   Откройте папку в браузере: http://localhost/Vet
   Безопасность: файлы с конфигурацией базы данных не включены в репозиторий.
